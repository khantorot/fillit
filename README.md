  <h3 align="center">Fillit</h3>

  <p align="center">
    Fillit is a project allowing you to discover and / or familiarize yourself with a recurring problem in programming: the search for an optimal solution among a very large number of possibilities, within a reasonable time. In the case of this project, it will be a question of arranging the Tetriminos between them and of determining the smallest possible edge which can accommodate them.
